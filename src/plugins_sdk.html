Plugins

Rodano highly modular design allows creation of plugins, which can be used to improve functionality.

Neurostatus EDSS consistency check

This plugin checks the consistency of the EDSS scores entered in the eCRF using the Neurostatus EDSS check API.

SDK

Rodano provides a publicly accessible REST API designed to support automation and integration tasks. Here are some common ways the API can be utilized:

  • Export data automatically
  • Workflow automation routines
  • External data import
  • User management
  • External systems connections (such as clinical trial management platforms)
  • Custom reports generation

For more information, refer to the r-rodano-sdk GitHub repository.
